# 🚀 ¿Qué es Rosita Finanzas?

Rosita Finanzas es un sistema web desarrollado para ayudar a peluqueras independientes a administrar su actividad financiera como contribuyentes de Primera Categoría.

El sistema no solo registra ventas.

Calcula automáticamente:

- Comisión del salón.
- Ingreso real de la peluquera.
- IVA.
- Flujo financiero.
- Historial mensual.
- Libro financiero.

Todo desde un Dashboard moderno desarrollado con React y Supabase.
