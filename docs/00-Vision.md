# Rosita Finanzas / Rosita ERP

## Visión

Rosita Finanzas nace para ayudar a una peluquera independiente que trabaja dentro de una peluquería a llevar un libro financiero claro, simple y profesional.

El objetivo no es solo registrar ventas, sino entregar una visión real del dinero disponible, impuestos, ingresos mensuales, gastos y crecimiento financiero.

## Contexto

Rosita trabaja en una peluquería bajo un modelo donde:

- El total vendido se divide 50% para la peluquería y 50% para Rosita.
- Rosita debe administrar su 50% como ingreso propio.
- Sobre el 50% correspondiente a Rosita se calcula IVA.
- La aplicación debe mostrar el mes actual por defecto.
- Cada mes debe poder revisarse de forma independiente.
- Los meses anteriores deben quedar disponibles como historial financiero.

## Misión

Transformar la información diaria de ventas, clientes, servicios, gastos e inventario en un libro financiero inteligente que ayude a tomar mejores decisiones.

## Principios

- Simple de usar.
- Visualmente atractivo.
- Pensado para peluqueras independientes.
- Enfocado en finanzas reales.
- Preparado para crecer como producto.
